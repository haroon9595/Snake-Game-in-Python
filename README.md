# Snake Game in Python

A classic **Snake Game** developed in **Python**. Control the snake, eat food to grow longer, and avoid colliding with walls or yourself. This project demonstrates fundamental programming concepts such as loops, conditionals, functions, and game logic implementation.

## Features

- Smooth snake movement
- Random food generation
- Score tracking system
- Game over conditions
- Simple and clean interface

## Requirements

- Python 3.x
- Pygame library (if used)
